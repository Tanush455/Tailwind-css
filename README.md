# Tailwind CSS Projects

This repository contains a collection of projects built with Tailwind CSS and Next.js. Each project is in its own subdirectory and demonstrates different features and use cases of Tailwind CSS.

## Projects

Here is a brief overview of the projects in this repository:

### 1. [tailwind-2](./tailwind-2)

A landing page for a fictional accounting and tax simplification service. This project is built with Next.js and features a rich component library based on [shadcn/ui](https://ui.shadcn.com/) and Radix UI. It showcases a modern and professional design.

**Features:**
- Next.js with App Router
- shadcn/ui components
- Tailwind CSS with custom theme
- Hero section with a call to action
- Navigation bar

### 2. [tailwind-3](./tailwind-3)

A simple landing page for a financial product with a "Join Waitlist" form. This project is a good example of a clean and minimalist design using Tailwind CSS.

**Features:**
- Next.js with App Router
- Minimalist design
- "Join Waitlist" form
- Gradient text effect

### 3. [tailwind-4](./tailwind-4)

A single-page component showcasing a card with a theme toggler. This project demonstrates how to implement dark mode and use icons with Tailwind CSS.

**Features:**
- Next.js with App Router
- Dark mode theme toggler using `next-themes`
- `lucide-react` for icons
- Animated icon marquee

### 4. [tailwind-6](./tailwind-6)

A page that showcases a 3D-transformed image with a hover effect. This project is a great example of how to create complex and interactive UI elements with Tailwind CSS.

**Features:**
- Next.js with App Router
- 3D transform and hover effects
- Theming with CSS variables
- Grid pattern background

### 5. [tailwindcss-1](./tailwindcss-1)

A landing page for an accounting and tax service, similar to `tailwind-2`. This project is a more basic version and does not use a large component library. It's a good example of building a landing page from scratch with Tailwind CSS.

**Features:**
- Next.js with App Router
- Custom-built components
- Hero section with a call to action
- Navigation bar

## Getting Started

To run any of these projects, navigate to the project's subdirectory and run the development server:

```bash
cd <project-directory>
npm run dev
```

For example, to run `tailwind-2`:

```bash
cd tailwind-2
npm run dev
```
